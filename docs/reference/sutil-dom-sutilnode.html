SutilNode Type

Namespace: Sutil
Assembly: Sutil.dll
Parent Module: DOM
Base Type: obj
All Interfaces: IEquatable<SutilNode> , IStructuralEquatable

SutilNode is a DOM node, the result of evaluating a SutilElement. Fragment and binding elements will return a GroupNode, which is a grouping of DOM nodes, so a SutilNode can also be a GroupNode. Finally, SutilNode can be an EmptyNode to represent the concept of None (or null)
